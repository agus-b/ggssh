# ggssh
ssh alias &amp; manager tools

# installation

```$ git clone https://github.com/agus-b/ggssh.git```


# how to?

```
$ ggssh add
SSH Alias []: domain
SSH User []: username
SSH IP/Host []: domain.com
SSH Identity Key []: /home/user/Documents/ssh_rsa
$ ggssh apply
$ . ~/.bashrc
```

# help
```$ ggssh help```
